3. ESPECIFICACIONES TÉCNICAS DE ADQUISICIÓN

Bandas espectrales descargadas:
  - B04 (Red): 665 nm, 10m resolución espacial
  - B08 (NIR): 842 nm, 10m resolución espacial

Formato de almacenamiento: GeoTIFF (FLOAT32)
Tamaño promedio por banda: ~25 KB (con resolución 512x512 px)
Tamaño máximo permitido: 10 MB por banda
Proyección: WGS84 (EPSG:4326)

Resolución espacial configurada:
  - Ancho: 512 píxeles
  - Alto: 512 píxeles
  - Resolución efectiva: ~1000 m/píxel (ajustada según tamaño de parcela)


4. INTEGRACIÓN CON INFRAESTRUCTURA COPERNICUS

APIs utilizadas:

1. STAC API (SpatioTemporal Asset Catalog)
   - URL: https://stac.dataspace.copernicus.eu/v1/search
   - Propósito: Consulta de metadatos (fechas, nubosidad, scene_id)
   - Autenticación: NO requerida
   - Método: POST con payload GeoJSON

2. Process API (Sentinel Hub)
   - URL: https://sh.dataspace.copernicus.eu/api/v1/process
   - Propósito: Descarga de bandas espectrales como GeoTIFF
   - Autenticación: OAuth2 Bearer Token (requerida)
   - Método: POST con evalscript JavaScript

3. OAuth2 Token Endpoint
   - URL: https://identity.dataspace.copernicus.eu/auth/realms/CDSE/protocol/openid-connect/token
   - Propósito: Obtención de access_token para Process API
   - Flujo: Backend Application Client (client_credentials)

7. ENDPOINTS REST IMPLEMENTADOS (OE1)

GET /api/sentinel/available-dates/{polygon_id}
  Query params: start_date, end_date, max_cloud
  Respuesta: Lista de fechas con cloud_cover y flag "acquired"
  Propósito: Consultar escenas disponibles para una parcela

POST /api/sentinel/acquire
  Body: { polygon_id, date, width, height }
  Respuesta: { acquisition_id, cloud_coverage, size_b04_kb, size_b08_kb }
  Propósito: Descargar y guardar bandas B04 y B08 en BD

9. CRITERIOS DE FILTRADO

Nubosidad máxima: 20%
Justificación: Venezuela (zona tropical) presenta mayor nubosidad que Europa.
  El criterio estándar (10-15%) resultaría en muy pocas fechas aptas.
  Se ajustó a 20% basándose en:
    - Recomendaciones para zonas tropicales (FAO, ESA)
    - Balance entre calidad de imagen y disponibilidad
    - Tests empíricos con parcelas reales del SRRG

Período de menor nubosidad en la zona:
  - Diciembre a marzo (temporada seca)
  - Mayor concentración de fechas aptas en estos meses

Nivel de procesamiento: Sentinel-2 Level-2A
  - Corrección atmosférica aplicada (Bottom of Atmosphere reflectance)
  - Corrección geométrica (ortorectificación)
  - Clasificación de nubes automática (máscaras incluidas)
